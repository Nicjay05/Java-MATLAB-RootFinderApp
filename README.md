🔢 Numerical Methods Solver

A modern Java & MATLAB application for solving Numerical Methods problems with interactive visualization and iterative analysis.

✨ Overview

The project combines:

🖥️ Java Swing for desktop GUI development
📊 MATLAB App Designer for mathematical computation and visualization
📈 Graph plotting for function analysis
📋 Iteration tables for step-by-step computation tracking

🚀 Features
📌 Root-Finding Methods

🔹 Bisection Method

🔹 False Position (Regula Falsi)

🔹 Newton-Raphson Method

🔹 Secant Method

🔹 Incremental Method

📈 Graph Visualization

📋 Iteration Tables

🛠️ Technologies Used:

Technology	Purpose
Java Swing	Desktop GUI Development
MATLAB App Designer	Mathematical Modeling & Visualization
Java	Core Application Logic
MATLAB	Numerical Computation
